Imagens dinâmicas

Você pode usar imagens dinâmicas em seu site para dar um visual diferente.

como criar imagens dinãmicas e para que servem?

As imagens dinâmicas servem para se enquadrarem em diferente tamanhos de dispositivo que o seu site for acessado. Ex. se o seu site for acessado por um smartfone, ela vai se encaixar na tela do smartfone. Se for acessado por um computador ou por uma tv ela (imagem dinâmicas) se encaxará na tela na qual ela for acessada

para criar uma imagem dinâmica, tem de se seguir os seguintes passos:

  1. usar a tag picture e dar um enter
  2. usar a tag imag e selecionar a maior imagem ( que deve ter um tamnho de 1050px)
  3. dentro da tag imag tem a tag alt. dentro desta tag preencha com imagem flexivel
  4. usar a tag source media type e preencher com max: para o primeiron source deve ter o temanho de 1050px e o segundo 750px
