Como adicionar fonte externa em css

Para adicionar fonte externa em css deve-se seguir os seguintes passos:

  • Usar a tag @font-face
  • Na tag URL deve ser preenchida com a fonte pretendida
  • Dar uma font-weight
  • Dar uma font-style
